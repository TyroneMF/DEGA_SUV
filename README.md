@DEGA_SUV
==============

![SUV Front View](https://raw.githubusercontent.com/deltagamer/DEGA_SUV/main/Release_Notes/20210214143044_1.jpg)

Features
=========
* Ability to open and close Rear and Turret doors
* Passenger FFV
* Bulletproof Glass
* Illuminating Dash
* No scripts for turret

Whats Planned
